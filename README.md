# Git Activity TBD1
This is a task to practice forking, cloning, staging, committing, and pushing.

## Task Instructions
1. Fork this repository (repo) into your own GitHub account
2. Clone the forked repo to your local machine
3. Find both instances of `.......` in `index.html` and replace with a word or short phrase of your choice
4. Stage, commit and push the changes
5. Verify your work
